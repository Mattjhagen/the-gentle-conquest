## Full Wrap Dimensions

```
Total Width: 12.284"
Total Height: 9.25"

Front Cover: 6" x 9"
Back Cover: 6" x 9"
Spine: ~0.284" (450 pages, cream paper)
Bleed: 0.125" on all edges
```

---

## Pixel Dimensions

```
Width: 3685 pixels
Height: 2775 pixels
Resolution: 300 DPI
Format: PDF (preferred) or high-res JPEG
Max File Size: 650 MB
```

---

## Spine Text

Your book has **~450 pages** → **Spine text IS allowed** (minimum 79 pages).

## Cover Layout

```
+------------------+------+------------------+
|                  |      |                  |
|   BACK COVER     | SPINE|   FRONT COVER    |
|                  |      |                  |
|  - Blurb text    | T    |  - Title         |
|  - ISBN barcode  | I    |  - Subtitle      |
|  - Author bio    | T    |  - Author        |
|                  | L    |  - Cover art     |
|                  | E    |                  |
|                  |      |                  |
+------------------+------+------------------+
     0.125" bleed on all edges
```
