## Typography Overlays

After generating the base image, add typography using design software:

### Title Treatment
```
Font: Helvetica Neue Light (or similar clean sans-serif)
Size: 72-96pt
Tracking: +100 to +200
Color: White or dark charcoal
Position: Upper third of cover
```

### Author Name
```
Font: Helvetica Neue Regular (or matching sans-serif)
Size: 24-36pt
Tracking: +50 to +100
Color: White or dark charcoal
Position: Bottom third of cover
```

## Color Grading Notes

### For eBook (RGB)
- Ensure colors are vibrant on screen
- Test on multiple devices (phone, tablet, e-reader)
- Check contrast at thumbnail size

### For Print (CMYK)
- Convert from RGB to CMYK carefully
- Check for color shift
- Request proof before final print

---

## Usage Rights

When using AI-generated images:
1. Check the terms of service for your AI tool
2. Most tools allow commercial use of generated images
3. Modify significantly to make unique
4. Consider hiring an artist for final production cover
5. AI can be used for concepts, not final product

---

## Recommended Workflow

1. Generate multiple concepts using AI prompts
2. Select strongest concept
3. Refine with design software (Photoshop, Figma, Canva)
4. Add typography and text elements
5. Test at multiple sizes
6. Commission final artwork if needed
7. Generate print-ready files
