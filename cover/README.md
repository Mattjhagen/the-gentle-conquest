# Cover Directory

This directory contains cover images and related assets.

## Files (to be created in Phase 5)

- `cover-ebook.jpg` — eBook cover (1600x2560 pixels, RGB)
- `cover-paperback.pdf` — Paperback cover (full wrap with spine)
- `cover-hardcover.pdf` — Hardcover cover (full wrap with spine)
- `cover-front.jpg` — Front cover only (for marketing)
- `cover-back.jpg` — Back cover only (for marketing)
- `cover-spine.jpg` — Spine only (for marketing)

## Specifications

### eBook Cover
- **Dimensions:** 1600 x 2560 pixels (minimum)
- **Aspect Ratio:** 1:1.6 (recommended)
- **Resolution:** 300 DPI
- **Color Space:** RGB
- **File Format:** JPEG or PNG
- **File Size:** Under 50MB

### Paperback Cover
- **Dimensions:** Full wrap (front + spine + back)
- **Bleed:** 0.125" (3mm) on all sides
- **Spine Width:** Calculated based on page count and paper type
- **Resolution:** 300 DPI
- **Color Space:** CMYK
- **File Format:** PDF

### Hardcover Cover
- **Dimensions:** Full wrap (front + spine + back)
- **Bleed:** 0.125" (3mm) on all sides
- **Spine Width:** Calculated based on page count and paper type
- **Resolution:** 300 DPI
- **Color Space:** CMYK
- **File Format:** PDF

## Design Notes

- Cover should convey: psychological tension, technology, human resistance
- Suggested imagery: abstract neural patterns, a single human figure, muted colors with one accent
- Title should be prominent and readable at thumbnail size
- Author name should be clear but not dominant

## Notes

- Cover design will be created in Phase 5
- Consider hiring a professional cover designer
- Test cover at multiple sizes (thumbnail, full size, print)
- Ensure text is readable at all sizes
